# LunchMate 🧡

**점심 한 끼로 시작하는 설렘** — 바쁜 직장인과 학생을 위한 소개팅 · 그룹 미팅 · AI 매칭 플랫폼.

Airbnb + Apple + Instagram 감성의 프리미엄 SaaS UI로 구현했습니다.

## ✨ 핵심 기능

- **점심시간 / 퇴근 후 1:1 소개팅** — 지역·시간·연령 필터 + 신청·결제 플로우
- **그룹 미팅** — 3:3 ~ 6:6, 실시간 모집 현황
- **AI 성향 매칭** — MBTI·관심사·라이프스타일 분석, 매칭 점수 & 공통 관심사
- **장소 자동 추천** — 제휴 카페·브런치·레스토랑
- **후기 & 성공 사례**, **마이페이지** (참여 이력 / 찜한 상대 / 결제 내역)

## 🛠 기술 스택

React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion · React Router

디자인: Gradient Mesh · Glass Morphism · Floating Elements · Parallax · Micro Interaction

## 🚀 개발

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 미리보기
```

## 📦 배포

`main` 브랜치에 푸시하면 GitHub Actions가 자동으로 GitHub Pages에 배포합니다.
